取出查询的结果集

  • fetch_all()：取出结果集中的所有数据，默认返回一个二维索引数组

  • fetch_array():取出结果集中的一条数据，并使用索引+关联数组方式返回

  • fetch_row():取出结果集中的一条数据，返回的是索引数组

  • fetch_object():从结果集中取得一行作为对象

  • fetch_assoc():返回结果集中的一条数据作为关联数组

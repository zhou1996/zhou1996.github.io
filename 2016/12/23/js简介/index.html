什么是JavaScript

javeScript是一种轻量级脚本语言呢

脚本语言：指的是它不具备开发操作系统的能力，而是只用来编写控制其他大型应用程序的“脚本”。

1.JavaScript的组成

  • ECMAscript:一套标准，最新的标准是ES6(简称)
    • 基本的语法结构：比如操作符、控制结构、语句
    • 标准库：Array、Date、Math等
  • BOM：Browser Object Model，浏览器对象模型
  • DOM: Document Object Model, 文档对象模型

2.变量

  1. 变量的声明和赋值是分开的2个步骤。
  2. 作用域：在函数内用var声明的变量为局部变量，在函数外声明的变量为全局变量。
  3. 重复声明的变量无效。

3.标识符

  • 标识符是用来识别具体对象的一个名称。
  • 最常见的标识符就是变量名，以及后面的函数名

标识符规则：

  1. 标识符命名必须保证不能以数字作为开始符号
  2. 标识符命名应当保证其具有见文识意的取名方式
  3. 自定义标识符不允许与关键字和保留名同名
  4. $符号应当写在标识符的首字母处
  5. js语言对标识符大小写敏感
